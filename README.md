# EcoRecicla

Aplicação de exemplo para gerenciar pontos de coleta e recompensas por reciclagem.

## Como rodar localmente

Pré-requisitos: Node.js (recomendo usar nvm) e npm.

```powershell
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
npm install
npm run dev
```

Abra `http://localhost:5173` no navegador.

## Tecnologias

- Vite
- TypeScript
- React
- Tailwind CSS
- Supabase

## Deploy

Use seu provedor preferido (Vercel, Netlify, etc.) para publicar o projeto. Configure as variáveis de ambiente do Supabase (`VITE_SUPABASE_URL`, `VITE_SUPABASE_PUBLISHABLE_KEY`) no painel do provedor.
